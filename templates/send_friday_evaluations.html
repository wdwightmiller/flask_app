How It Works
  1. System finds all active rotation blocks (blocks that include today's date)
  2. Collects all assignments within those blocks marked for Friday sending
  3. Sends each fellow their assigned survey via text message
  4. Tracks what was sent to prevent duplicate messages
  5. Logs everything in the tracking page
Friday Schedule

Recommended Workflow:

  • Thursday: Review assignments for active rotation blocks
  • Friday Morning: Come to this page and preview what will be sent
  • Friday (9-10 AM): Click "Send All Evaluations"
  • After Sending: Check tracking page to confirm all sent successfully

Tip: You can send evaluations any day, but Friday is recommended to match your 2-week rotation schedule.

Before Sending

  • Created rotation blocks for the current 2-week period
  • Added all necessary surveys
  • Assigned fellows to their surveys for the current block
  • Verified fellow phone numbers are correct
